Important FAQ

What is the Document Object Model(DOM)?

DOM is a programming interface for web documents. It represents the structure of a web page, allowing scripts to interact with, manipulate, and modify the document's content, structure, and style.It represents the document as a tree of nodes, where each node is an object representing part of the document.

How do you select element from the DOM?

We can select elements by taking id, class, and tag name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a pattern in the DOM where we attach a single event listener to a parent element to handle events triggered by its child elements, rather than attaching individual event listeners to each child. This is useful for managing events efficiently, especially when dealing with dynamically added elements or a large number of child elements.
